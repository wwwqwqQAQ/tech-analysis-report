# Tech Industry Analysis Daily Report

科技产业分析日报生成器 — 覆盖 AI/芯片/新能源/互联网/航天/政策/投融资七大赛道，以技术成熟度曲线和产业经济学框架进行深度分析。

## Features

- **7 大科技赛道** — AI/大模型、半导体/芯片、新能源/电动车、互联网/平台、航天/硬科技、科技政策、投融资
- **企业竞争格局** — 追踪华为/英伟达/特斯拉/字节等 18 家核心科技企业的媒体关注度
- **学术分析框架** — Hype Cycle 技术成熟度曲线、波特五力竞争模型、技术采用生命周期
- **产业趋势研判** — AI Agent 新范式、芯片双轨化、新能源洗牌、科技监管精准化
- **图表可视化** — 赛道分布、企业关注度排行、热度时间线

## Quick Start

```bash
pip3 install python-docx matplotlib numpy
python3 generate_tech_report.py
```

Output: `~/Desktop/项目/科技报告/科技产业分析_YYYY-MM-DD.docx`

## Report Structure

| Section | Content |
|---------|---------|
| 核心发现 | 当日科技产业全景概览 |
| 一~七 | AI/芯片/新能源/互联网/航天/政策/创投 各赛道新闻+深度分析 |
| 八 | 技术趋势深度研判（跨赛道交叉分析） |
| 九 | 科技产业前瞻 |

## Methodology

- **Hype Cycle** — 判断各技术赛道所处阶段（触发期/膨胀期/幻灭期/爬升期/成熟期）
- **Porter's Five Forces** — 产业竞争格局分析
- **Technology Adoption Lifecycle** — 技术扩散阶段判断
